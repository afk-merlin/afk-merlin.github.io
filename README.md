# afk-merlin.github.io

Personal portfolio for **Trafalgar D. Merlin** (*AFK Merlin*) — drone, multirotor & IoT engineer. A **Trafalgar Law / Worst Generation** themed One Piece immersive web experience. ☠️

> Cloned from the Backbenchers Pirates engine (the Zoro-vibe build) and re-forged in **Law colours**: cyan "ROOM" + Heart-Pirates yellow on tactical navy, with a nautical-blueprint light mode.

## 🚀 Live Site

[https://afk-merlin.github.io](https://afk-merlin.github.io)

## ☠️ Theme & Features

- **Trafalgar Law palette** — cyan (ROOM) `#22d3ee` + Heart-Pirates yellow `#facc15` on tactical navy; light mode is a cool nautical **blueprint** (not warm parchment)
- Shared **cyber Jolly Roger** (Backbenchers Pirates) in nav, footer, flags & favicon
- Merlin's **WANTED poster** hero with parallax + shine, his profile photo
- Immersive engine: "Engaging ROOM" preloader, film-grain + vignette, particles, sword-slash cursor, typewriter, Operations timeline
- Sections: Dossier · Operations · Loadout · Dispatches · Hail
- Loadout framed as Law abilities → **ROOM** (drones/multirotor), **Shambles** (IoT/gadgets), **Scan** (telemetry/comms), **Scalpel** (embedded/firmware), **Tact** (field/recon)
- Light/Dark toggle (persisted), `prefers-reduced-motion`, mobile responsive

Name/alias, Operations, Dispatches, contact email & links, social `og:*` URLs.
